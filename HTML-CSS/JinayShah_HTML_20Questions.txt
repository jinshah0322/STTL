1.What is difference between Tag and Element in HTML?
Tag refers to the individual HTML code, such as <p> or <h1>
Element consist of opening tag, content inside the tag and closing tag such as <p>Jinay shah</p>

2. Which are void elements in HTML?
Void element are those which do not have closing tag such has <br>,<img>

3. What is semantic code in HTML?
Semantic code adds meaning to the structure, helping both browsers and developers understand the content's purpose. Examples include <header>, <footer>

4. What is the difference between pseudo class and pseudo elements in CSS?
Pseudo Class Selects elements based on their state, like :hover or :first-child.
Pseudo Element Selects and styles parts of an element, like ::before or ::after.
